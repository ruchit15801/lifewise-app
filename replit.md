# SpendIQ - UPI Expense Intelligence App

## Overview
A mobile-first personal finance app that analyzes UPI/SMS transactions to provide spending insights, money leak detection, bill alerts, and monthly financial summaries. Built with a futuristic dark UI inspired by CRED and Jupiter.

## Tech Stack
- **Frontend**: Expo (React Native) with Expo Router for file-based routing
- **Backend**: Express.js (Node.js/TypeScript)
- **State Management**: React Context + AsyncStorage for local persistence
- **UI Framework**: React Native StyleSheet with Inter font family
- **Animations**: react-native-reanimated for entrance animations

## Project Structure
```
app/
  _layout.tsx          # Root layout with providers (ExpenseProvider, QueryClient)
  (tabs)/
    _layout.tsx        # Tab navigation (5 tabs with liquid glass support)
    index.tsx          # Home Dashboard
    transactions.tsx   # Transaction Timeline
    reports.tsx        # Monthly Report
    leaks.tsx          # Money Leak Detector
    bills.tsx          # Bill Reminders
lib/
  data.ts              # Types, mock data generators, utility functions
  expense-context.tsx  # React Context for expense state management
  query-client.ts      # React Query client config
constants/
  colors.ts            # Dark futuristic theme colors
components/
  ErrorBoundary.tsx    # Error boundary component
  ErrorFallback.tsx    # Error fallback UI
server/
  index.ts             # Express server entry
  routes.ts            # API routes
```

## Key Features
1. **Home Dashboard** - Daily/monthly spend summary, budget tracking, category breakdown, recent transactions
2. **Transaction Timeline** - Chronological transaction list with category filters, grouped by date
3. **Monthly Reports** - Month-by-month spending analysis with category bars, top merchants, savings rate
4. **Money Leak Detector** - Identifies recurring expenses, estimates monthly/yearly savings potential
5. **Bill Reminders** - Upcoming bill tracking with urgency indicators, mark-as-paid toggle

## Design
- Dark futuristic theme (#06060E background)
- Neon green accent (#00E5A0) inspired by CRED
- Glassmorphic cards with gradient backgrounds
- Category color coding system
- Inter font family (Regular, Medium, SemiBold, Bold)
- NativeTabs with liquid glass support for iOS 26+

## Data
Uses demo transaction data generated by `lib/data.ts`. Transactions simulate realistic Indian UPI payments across categories (Swiggy, Zomato, Amazon, Uber, etc.). Data persists locally via AsyncStorage.
